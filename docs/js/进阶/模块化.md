# 模块化



## 前言



## CommonJS规范

服务器使用。

Node中实现了CommonJS的规范，但慢慢的这个规范被ES6的模块化所取代。

关键字：module.export、exports

## AMD规范

浏览器端使用。

异步加载。

require.js实现了这个规范。



## CMD规范

浏览器端使用。

用同步的方式去写代码。

sea.js实现了这个规范。



## ES6模块

服务端可以直接使用。

两个关键字：export和import
